# cameliaressam.github.io

Personal academic portfolio website — built with plain HTML, CSS, and JavaScript. Deployed via GitHub Pages.

## Pages

- **Home** (`index.html`) — Introduction, about section, and quick links
- **CV** (`cv.html`) — Education, experience, skills, and languages
- **Publications** (`publications.html`) — Research papers with abstract toggle, BibTeX copy, and year filter

## Getting started

1. Replace all `<!-- PLACEHOLDER -->` sections with your actual content
2. Add your profile photo to `assets/` and update the `<img>` tag in `index.html`
3. Update social links (GitHub, LinkedIn, email, Google Scholar) across all pages
4. Optionally add a PDF of your CV to `assets/` and update the download link in `cv.html`
5. Push to `main` — GitHub Pages will serve the site automatically

## Structure

```
├── index.html          # Home page
├── cv.html             # CV page
├── publications.html   # Publications page
├── css/
│   └── style.css       # Shared styles
├── js/
│   └── main.js         # Interactions (nav, filters, BibTeX)
└── assets/             # Images, PDF, favicon
```
